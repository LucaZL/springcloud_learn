读取配置文件三种方式：

1. Environment

    @Autowired  
    private Environment env;

    String port = env.getProperty("server.port")

2. @Value  
    @Value("server.port")  
    private String port;

3. 自定义配置类  
    @ConfigurationProperties(prefix = "user.zl")  
    @Component  
    @Data
    public class MyConfig {

        private String name;  
    }

    使用：  
    @Autowired  
    private MyConfig myConfig;

    myConfig.getName();

    application.properties  
    user.zl.name=baihaibao
